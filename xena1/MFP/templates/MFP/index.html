About



Xena.AI,Advanced feedback system. Any service is of no use without feedback. We provide you with an easy way to create your own feedback portal be it any kind of service you provide.

The idea is to provide you with facilities like comparisons of various types of data using graphs, periodic reports and an interactive user interface to collect user feedback. We help you find the last missing piece of your puzzle:)




Freqently Asked Questions

What is Xena.AI ?

Xena is a feedback portal which records users feedback using hand gestures and provide a detailed analysis on the compiled data.

Why should we use Xena? ?

User feedbacks help to improve your service and rectify your product. Xena is a perfect tool to do so.

What technologies do we use?

We use openCV library to record hand gestures and Django for all the backend.

Who all can use?

Any service which requires feedback can use Xena.AI
